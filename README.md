### GLOBAL DEPEDENCIES
 - NodeJs 6.9.5
   
   ``` 
   $ curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
   $ sudo apt-get install -y nodejs
   ```
 
### DEPEDENCIES
 - Install [yarn](https://yarnpkg.com/en/) and execute:
  
  ```
  $ sudo npm install -g nodemon
  $ sudo npm install -g karma-cli
  ```

### Run app:
yarn start

### Run test:
yarn test
